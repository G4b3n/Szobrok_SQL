Budapesti Szobrok Adatbázis Projekt

Órai SQL feladat HTML-es vizualizálása

1Margit

Feladat: Soroljuk fel lekérdezés segítségével ábécérendben a Margitsziget szobrainak címét és a felállításuktól eltelt időt.

Megoldás

SELECT cim as "Neve", hely as "Helyszín", 2026-ev as "Szobor kora" 
FROM szobor
WHERE hely = 'Margitsziget'
ORDER BY cim ASC;

2Legtobb

Feladat: Melyik évben állították fel a legtöbb szobrot, és ez hány alkotást jelent?

Megoldás

SELECT ev AS "Év", COUNT(*) AS "Szobrok száma"
FROM szobor
WHERE ev IS NOT NULL
GROUP BY ev
ORDER BY COUNT(*) DESC
LIMIT 1;

3Vegyes

Feladat: Hány olyan vegyes technikájú szobor van, amely egyszerre tartalmaz mészkövet és bronzot?

Megoldás

SELECT COUNT(DISTINCT szoborid) AS "Vegyes szobrok száma"
FROM anyag
WHERE nev = 'mészkő'
  AND szoborid IN (
    SELECT szoborid FROM anyag WHERE nev = 'bronz');
